# WeeklyNews 每周全球时事新闻简报

每周一早上 08:00（北京时间）自动抓取全球各领域新闻，生成中英双语 Word 文档并推送至本仓库。

## 涵盖板块

| 板块 | 来源 |
|------|------|
| 🌍 国际政治 | BBC World · Reuters |
| 💰 经济财经 | Reuters Business · BBC Business |
| 🔬 科技科学 | BBC Technology · BBC Science |
| 🎭 文化社会 | BBC Entertainment · Reuters Lifestyle |
| ⚽ 体育 | BBC Sport |

## 报告存放位置

生成的 Word 文档存放于 [`reports/`](./reports/) 目录，文件名格式：`WeeklyNews_YYYY-MM-DD.docx`

## 手动触发

在 GitHub 仓库页面 → Actions → Weekly News Generator → Run workflow
